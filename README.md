# e-Shop - Full Stack MERN E-Commerce Platform

e-Shop is a fully functional, modern e-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). It features a sleek user storefront for browsing clothing items, a dynamic shopping cart, and a dedicated Admin Panel for product management.

## 🚀 Features

### Frontend (User Storefront)
* **Modern UI/UX:** Clean, responsive design optimized for desktop and mobile devices.
* **Product Catalog:** Filter products by categories (Men, Women, Kids).
* **Product Details:** Detailed view with image zoom, size selection, and description.
* **Shopping Cart:** Add, remove, and update item quantities in real-time.
* **User Authentication:** Secure signup and login functionality using JWT.

### Backend & Admin Panel
* **Admin Dashboard:** Dedicated interface to manage the store's inventory.
* **Product Management:** Add new products (with image upload) and remove existing ones.
* **Secure API:** RESTful API routes protected with middleware.
* **Database Integration:** MongoDB used for storing product data and user credentials.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, CSS3, Hooks, Context API
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Authentication:** JSON Web Tokens (JWT) & bcryptjs
* **Image Storage:** Multer (Local Storage / Cloudinary integration)

Follow these steps to run the project locally on your machine.
